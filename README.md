# FPGA HFT Opportunities Tracker

Automated tracker for FPGA / DSP / Signal Processing internships, co-ops and
graduate programs at ~60 HFT & quant firms. This file is regenerated every
12 hours by GitHub Actions — the first workflow run will replace this
placeholder with live listings, statistics, and per-company source status.

Setup: see [setup_instructions.md](setup_instructions.md).
